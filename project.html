About the Project

I have work on projects are Disease Prediction & Doctor Recommendation using Machine Learning and flask , Airline Reservation System using Python(Flask),Mysql connectivit and on Credit Card Fraud Detection using Machine Learning. The system helps users identify health conditions and detects fraudulent transactions in real time.

🩺 Disease Prediction & Doctor Recommendation

  • Users enter their symptoms, age, and medical history.
  • The system analyzes symptoms and suggests possible diseases.
  • It recommends specialist doctors based on the predicted condition.
  • Users can find contact details of doctors near their location.

🧠 Machine Learning Algorithm Used:

  • 🔹 Decision Tree Algorithm

✈️ Airline Reservation System : SkyWay

This web-based system allows users to search and book flights with real-time data handling, ensuring a smooth travel booking experience. It also offers admin controls for managing flights and crew.

Key Functionalities:

  • 🔎 Flight search by source, destination, and date
  • 🛫 Multi-passenger booking with PNR generation
  • 📅 View flight schedules and itinerary details
  • ❌ Flight and itinerary cancellation
  • 💻Secure login/signup for personalized experience

💻 Technologies Used:

  • 🔹 Backend: Python (Flask), MySQL
  • 🔹 Frontend: HTML, CSS
  • 🔹 Features: Real-time flight search, authentication

🔍 Credit Card Fraud Detection Model

The fraud detection model helps in identifying fraudulent transactions using machine learning algorithms.

How It Works?

  • Collects transaction data including amount, location, time, and frequency.
  • Applies Machine Learning Algorithms to detect unusual patterns.
  • Flags suspicious transactions for further review.

🧠 Machine Learning Algorithm Used:

  • 🔹 Logistic Regression

  • 📌 Credit card fraud detection
